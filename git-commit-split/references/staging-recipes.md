# Staging Recipes

- Move into src:
  git add -A -- src bot.py buttons.py Helper.py main.py test.py

- Docker entrypoint:
  git add Dockerfile

- Docs update:
  git add README.md AGENTS.md

- Remove tooling:
  git add -A -- .claude .gemini .specify

- Remove workflows:
  git add -A -- .github/workflows
